= Smartis.API PHP client
Небольшая библиотека для работы с базовыми методами Smartis.API
Smartis - система сквозной аналитики с поддержкой бизнес решений для недвижимости

Smartis.API позволяет обновлять данные в системе из внешних сервисов и проектов, а также предоставляет доступ к отчетности создаваемую внутри системы для внешних сервисов.
Например, можно вставить в CRM клиента готовый дашборд по аналитике рекламных каналов.
